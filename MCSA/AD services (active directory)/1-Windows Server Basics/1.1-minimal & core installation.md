- minimal core installation does not contain GUI ,, U manage server using CMD (command prompt) & power shell & Remote management tools like windows admin .

# why network admin prefer core ? 
1-Reduce attack surface --> since the GUI  ,, there are fewer files and services for Hackers to exploits .
2-Lower resource consumption .
3-fewer updates .

- first thing u do after installation is typing sconfig in the terminal ,, this is a text based menu to perform basic task .

- to change computer name -->choose option 2 to rename the server .
- to change Network settings --> change option 8 to set static ip address .
- to change remote management -->choose option 4 to enable management from your physical pc .

# powershell  facilities :

1-check ip configuration : 
```powershhell
Get-Netowkipconfiguration
```
![[../../../attachments/Pasted image 20260312034749.png|Pasted image 20260312034749.png]]

2-set a static ip  :
```powershell 
New-NetIPAddress -InterfaceAlias "NIC1" -IPAddress 192.168.1.10 -PrefixLength 24 -DefaultGateway 192.168.1.1
```
![[../../../attachments/Pasted image 20260312035818.png|Pasted image 20260312035818.png]]
3-install role : 
```powershell
install-windwosfeature -name dns -includemanagementtools 
```
