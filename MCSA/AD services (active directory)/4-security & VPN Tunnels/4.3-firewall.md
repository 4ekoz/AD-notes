# 1- ports 
- computer have many ports  (from 0 to 65535) .
- well known : from 0-1023 (for services)  ,,ex : port 80 for internet , port 53 for DNS ,, etc.
- user-defined : from  1024 to 65535 that computer use to open service with external server .
# 2- HOW  firewall take a decision  ? 
- when packet come to firewall  ,, firewall see 4 things : sip , dip , port , mac address .. 
- if a firewall has rule like "prevent any one from port 80 " .,, any pc can't pass .
