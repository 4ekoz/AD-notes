- But imagine your "Zagazig" branch has a very slow 56kbps link to "Cairo." Trying to sync the entire Active Directory database over that slow connection would take days and likely fail.
**IFM** is the solution to this.
- **IFM** allows you to "export" a snapshot of the Active Directory database ($ntds.dit$), the transaction logs, and the SYSVOL folder onto a USB drive or external media. You then physically take that media to the new site and use it to install the new DC locally, rather than downloading everything over the VPN.
