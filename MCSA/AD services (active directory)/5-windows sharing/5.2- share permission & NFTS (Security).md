# share permission 
- Only when accessing a folder **over the network** (using a path like `\\Server\Share`).
- DATA are stored In the **Windows Registry** .
- Very basic. You only have three options: **Read**, **Change**, and **Full Control** ,,  They **do not protect** the files if someone logs in locally (sits physically at the server).

#  NTFS (new technology file system) permissions : 
- Everywhere. Whether the user is local (at the server) or coming from the network .
- data store directly on the file system .
- very detailed. You can control specific actions like "List Folder Contents," "Write," "Delete," or "Modify Permissions.
- Permissions can flow down from a parent folder to all subfolders and files inside it.
