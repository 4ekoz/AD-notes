- RDP is a proprietary protocol developed by Microsoft which provides a user with a graphical interface to connect to another computer over a network connection.

-  **Default Port:** TCP/UDP **3389**.

- _If I have a username and password, I can RDP into any machine."_ **In Active Directory, this is FALSE.**

- the permission of user : allow specific user use RDP .
