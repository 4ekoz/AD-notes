after that we going to : 
1- **Select Deployment Operation:** "Add a new domain to an existing forest" .
2-  **Select Domain Type:** Child Domain.
3- **Parent Domain Name:** `cisco.com`.
4 - **New Domain Name:** `cita` (This will make the full name `cita.cisco.com`).
5 - **Credentials:** Click "Change" and type `cisco.com\Administrator` and the password for the Parent server.


Nowww ,,,  "Parent & Child" trust is created. You can now go to **Active Directory Domains and Trusts** on the Parent and see the child listed. .
