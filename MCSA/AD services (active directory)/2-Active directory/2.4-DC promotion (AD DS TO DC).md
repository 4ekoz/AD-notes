# what is DC promotion ? 
process of installing AD DS role on windows server and then promoting it to become Domain controller ,, this creates a new domain , new forest or adds replica to an existing one . 


# install the role : 

```powershell 
install-windowsfeature -name ad-domain-services -includemanagementtools
```
# promotion server to Domain controller : 
After installing the role, you will see a **Yellow Warning** in Server Manager. Click **"Promote this server to a domain controller"**. You have 3 options:
- **Add a domain controller to an existing domain:** (Replica for Redundancy).
    
- **Add a new domain to an existing forest:** (Child Domain).
    
- **Add a new forest:** (The first DC in the whole network - **This is what you will choose**).


# important configuration : 
- **Root Domain Name:** (e.g., `shreef.local` or `cita.local`).
    
- **Forest/Domain Functional Level:** choose the lowest windows server available in the network .
    
- **DSRM Password:** (Directory Services Restore Mode) : this is password that i use if AD dropped and making recovery for it . 



#note : the first DC created in the forest hold all the FSMO  ( the 5 master roles ) ,, if the server goes down ,, the whole network has "heat attack" ,, in pentesting owing the DC mean u own the entire kingdom .
